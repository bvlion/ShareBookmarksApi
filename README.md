# ShareBookmarksApi

[ShareBookmarks](https://github.com/bvlion/ShareBookmarks) 用 API 

## FW

- [Ktor](https://ktor.io/)
- [Exposed](https://github.com/JetBrains/Exposed/wiki/DSL)
- [jjwt](https://github.com/jwtk/jjwt)

## 環境構築

### インストール

- IntelliJ
- Docker for Mac

### 実行

- `docker-compose up -d` でローカル DB 起動
- `gradle -> application -> run`
